Patch release on top of [v1.0.0](https://github.com/csiro-fair/marimba/releases/tag/v1.0.0).

## Fixed

- Empty file-type entries (null or blank values) are now filtered out when generating lists of unique file types in dataset summaries.
- `ifdo` upper bound tightened from `<2` to `<1.3` to ensure compatibility with the published package.
