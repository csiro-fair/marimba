Marimba v0.3.0 substantially expands the initial prototype, establishing the core project/collection/pipeline/dataset/target architecture across over 130 changed files.

**Highlights:**
- Distribution targets for AWS S3 and DAP servers
- Pipeline repository management via `gitpython`
- Geographic summary map generation via `staticmap`
- EXIF metadata writing via `piexif`
- Initial `pytest` test suite and pre-commit hook configuration

## Dependencies

Added: `gitpython`, `boto3`, `staticmap`, `piexif`, `psutil`, `pydoit`  
Removed: `cookiecutter`  
Updated: `ifdo` to 1.1.2
